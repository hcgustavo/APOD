About

What it is

NASA's Picture of the Day is a simple website to show each day a different image or video of our beautiful universe chosen by NASA itself. Accompanying each picture, there is a description with more information on the subject of the media.

You can also have access to the media archive by choosing a specific date on the date picker located at the center top of the home page.


How it works

This website uses an API called APOD (Astronomy Picture of the Day), provided by NASA, that allows us to request a picture from their database as well as some other information like the description of the media.

For more information on the APOD API, please check NASA Open APIs website: api.nasa.gov/api.html#apod
